# 高齢親安否確認アプリ 仕様書

## 概要

高齢の親に「今日の気分」をタップしてもらうだけで安否確認ができるシンプルなWebアプリケーション。

## 目的

- 高齢者が毎日1タップするだけで、家族に「今日も元気」と伝えられる
- 操作を極限までシンプルにし、高齢者でも迷わず使える
- キャラクターからの温かいメッセージで、日々の利用を楽しくする

## 機能一覧

### 1. 気分選択画面 (`/`)

- 3つの気分ボタンを表示
  - 「幸せだった」（happy）
  - 「普通」（normal）
  - 「ちょっと悲しい」（sad）
- 各ボタンにはイラスト画像を表示
- タップするとキャラクター結果画面に遷移

### 2. キャラクター結果画面 (`/result/:mood`)

- ランダムに選ばれたキャラクターが表示される
- 気分に応じたメッセージを表示
- **音声読み上げ機能**: Web Speech API を使用して自動再生
- 「もう一度」ボタンで気分選択画面に戻る

## キャラクター

|----|------|------|----------|

| fortune | 謎の占い師 | 意味不明で回りくどい | 最も低く、遅め |
